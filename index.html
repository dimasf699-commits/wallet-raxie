Konfigurasi Firebase
Ikuti langkah berikut agar data tersinkron untuk semua pengguna:
1
Buka console.firebase.google.com → buat project baru
2
Di project, klik Add app → pilih ikon Web (</>) → daftarkan app
3
Copy nilai apiKey, authDomain, databaseURL, projectId ke bagian FIREBASE CONFIG di kode HTML ini
4
Di Firebase Console → Realtime Database → Create Database → pilih region → klik Rules → ubah ke: { "rules": { ".read": true, ".write": true } } → Publish
5
Simpan file dan buka kembali di browser
⚠ Rules di atas bersifat publik (semua orang bisa baca/tulis). Cocok untuk penggunaan internal. Untuk produksi, tambahkan autentikasi.
