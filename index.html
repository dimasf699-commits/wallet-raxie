Konfigurasi Firebase
Ikuti langkah berikut agar data tersinkron untuk semua pengguna:
1
Buka console.firebase.google.com → buat project baru
2
Klik Add app → pilih Web (</>) → daftarkan app
3
Copy apiKey, authDomain, databaseURL, projectId ke bagian FIREBASE CONFIG di kode
4
Realtime Database → Rules → { "rules": { ".read": true, ".write": true } } → Publish
5
Simpan dan buka kembali
